# Основные команды Git

- **git init** - *инициализация локального репозитория*
- **git status** - *получить информацию от git о его текущем состоянии*
- **git add** - ~~добавить файл или файлы к предыдущему коммиту~~
- **git commit -m "message"** - создание коммита
- **git log** - вывод на экран истории всех коммитов с их хеш-логами
- **git checkout** - переход от одного коммита к другому
- **git checkout master** - вернуться к актуальному ссотоянию и продолжить работу
- **git diff** - увидеть разницу между текущим файлом и закоммиченным файлом
- **git clone <ссылка>** - скопировать удаленный репозиторий
- **git push** - загрузить репозиторий в GitHub
- **git pull** - скачать репозиторий из GitHub
